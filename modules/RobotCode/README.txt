This folder contains the current code to control the robot

Contents:
--Robot.py:
	Contains a class that acts as the robot. It handles the communication the robot. This class can be instantiated and functions can be called on it to move the robot.

--RobotController.py
	Contains the a basic gui and the ability to control the robot with key strokes. To run, click on it (Python Launcher should open and run). If that doens't work, navigate to the folder using a terminal and type "python RobotController.py"

	Note: In order to control the robot, you must have the window in focus (click on it)
